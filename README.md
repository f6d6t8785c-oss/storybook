# My story

A simple multi-page story website built with HTML and CSS.

## Pages

- `index.html` (home page)
- `chapter1.html`
- `chapter2.html`
- `chapter3.html`
- `chapter4.html`

## How to run

Open `index.html` in your browser.

## Project structure

- `story.css` contains the styling for the pages.
- Image assets are stored in the project root and used by the HTML pages.
